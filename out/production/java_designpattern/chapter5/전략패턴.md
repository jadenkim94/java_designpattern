## 전략패턴(Strategy pattern) 이란?
1. 동일 계열의 알고리즘(=행위)들을 정의하고
2. 각 알고리즘을 캡슐화하여
3. 이 알고리즘들을 상호교환 가능하게 하는 것 


```java
public class SoccerPlayer {

    private String name;

    public SoccerPlayer(String name) {

    }

    public void shoot(){

        } else if ( name.equals(TECHNICAL)){

        }
    };

}
```
